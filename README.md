# HR-Dashboard

Project Title -
HR Data Analytics Dashboard | Employee Attrition Insights (Power BI)

Short Description -
This project presents an HR Data Analytics Dashboard developed using Power BI, focusing on analyzing employee attrition and workforce patterns.The dashboard provides key insights into employee behavior, helping organizations understand why employees leave and how different factors impact attrition.

Insights Covered -
Attrition by Education – Identifies which educational backgrounds have higher attrition
Attrition by Age Group – Highlights age segments with maximum employee turnover
Attrition by Salary – Shows how salary ranges impact attrition rates
Attrition by Job Role – Compares attrition across roles like Sales Executive & Manager
Attrition by Satisfaction Level – Analyzes how job satisfaction affects employee retention
Years of Service Analysis – Tracks attrition trends over time

Technologies Used -
Microsoft Power BI Desktop – For building interactive dashboards
DAX (Data Analysis Expressions) – For creating measures like Attrition Rate, Employee Count
Power Query – For data cleaning and transformation
Data Modeling – Creating relationships between HR tables (Employee, AgeGroup, Gender, etc.)
Excel Dataset – Source data for HR analytics

Purpose of the Project -
Identify why employees are leaving the company
Analyze attrition based on age, salary, job role, and education
Track key HR metrics like attrition rate, average salary, and tenure
Support HR teams in making data-driven decisions
Improve employee retention strategies
